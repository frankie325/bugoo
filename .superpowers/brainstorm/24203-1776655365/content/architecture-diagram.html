关键设计决策

为什么翻译用 DeepL 而不是本地词库？

通用人群需求多语言，部署本地词库体积大，DeepL 免费额度（500万字符/月）足够个人用户，且翻译质量更高。

为什么艾宾浩斯放 Rust 层而不是前端？

复习时间计算是后台调度任务，Rust 定时器比 JS 更稳定可靠。前端只负责 UI 展示，不负责计算。

为什么不自己实现云同步？

极简产品不做账户系统。云同步委托给 iCloud/Dropbox，系统已经处理好了冲突和版本管理，我们只需读写文件。
