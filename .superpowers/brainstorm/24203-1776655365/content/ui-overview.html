三个核心视图

1. 划词翻译浮窗

用户划词 → 浮窗显示翻译 → 一键"添加到生词本"

只有一个输入框的视觉，其他都隐藏

2. 生词本列表

单词 + 释义 + 复习状态 + 下次复习时间

没有复杂标签系统，没有分类，只有"复习中/已记住"

3. 复习通知卡片

通知横幅直接显示单词 + 释义 + 认识/不认识 按钮

无需打开App，在通知层完成整个复习闭环

极简设计的核心：用户不需要思考"该从哪里开始"
