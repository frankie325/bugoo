设计更新：已加入新功能

以下是根据你的反馈做出的调整

新增 / 调整的功能

单词发音（TTS）

使用 macOS NSSpeechSynthesizer / Windows SAPI 系统级 TTS，无须第三方音频服务，不额外消耗 API 额度。

划词浮窗和复习通知中都有发音按钮，默认不自动播放（避免打扰他人）

标签系统

扁平标签（如 CET4、经济、日常），不设文件夹层级。生词列表可按标签筛选，标签默认折叠显示不干扰阅读。

新增 tags 表管理标签名和颜色

笔记功能

每个单词可以附加一条简短笔记，仅在点击展开单词详情时才显示，不在列表中喧宾夺主。

笔记字段在 words 表中为可空字段
