# 划词弹窗功能

[] popup高度自动跟随内容高度进行变化，最大高度为600。监听中间内容高度变化，前端使用ResizeObserve，去看有没有相关的工具库使用
[] 如果正在翻译中，用户关闭弹窗时需要取消翻译请求
[x] 在学习-划词设置中，添加划词字符长度限制设置，默认200个字符。
[] 如果是已存在数据库表的单词，标签tooltip在关闭tooltip时需要判断，与tooltip打开时相比较，标签的选中项是否有变化，如果变化了，则更新该单词的标签
[] 修复关闭弹窗时会跳转到主面板
[] 自动关闭需要读取设置判断
[] 前端根据翻译返回的状态显示不同的提示
[] 设置面板添加打开弹窗时自动复制
[] words表删除tags字段
# 翻译功能
[] 识别非单词时，只需要翻译translation字段就行了，其余字段返回为空。
[] 单词词库弄一个单独的服务，不带在应用里面了，服务查询结果进行统一。
