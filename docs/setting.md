# Bugoo 设置页交互说明（Settings Page Spec）

版本：v1.0  
优先级：P0（核心配置模块）  
适用：前端 / 客户端 / 产品 / 测试

---

# 0. 页面定位

设置页用于：

- 管理学习策略（复习算法、频率）
- 控制通知行为（提醒频率、时间段）
- 管理数据（单词本、标签）
- 配置系统行为（划词、快捷键等）

---

# 1. 页面结构

---

## 整体布局

SettingsPage
├── Sidebar（设置分类）
├── Content（设置详情）
